MetaWards.iterators¶

These are the iterator functions that are used by the model to advance the outbreak from day to day. The functions divide into two main types, the most important of which are described below;

iterate_functions which control which functions are called to advance the outbreak
  • iterate_default()
  • iterate_custom()
  • iterate_weekday()
  • iterate_weekend()
  • iterate_working_week()
advance_functions which perform the actual work of advancing the outbreak
  • advance_foi()
  • advance_additional()
  • advance_infprob()
  • advance_fixed()
  • advance_play()
  • advance_work()

All of the above functions (and the many others in metawards.iterators) are described in more detail here;
