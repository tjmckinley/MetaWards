MetaWards.movers¶

These are the mover functions that are used to move individuals between different demographics during a multi-demographics model run.

The functions divide into two main types;

move_functions that determine which functions are called to move individuals
  • move_default()
  • move_custom()
go_functions that perform the work of moving individuals between demographics
  • go_isolate()

All of the above functions (and the many others in metawards.movers) are described in more detail here;
