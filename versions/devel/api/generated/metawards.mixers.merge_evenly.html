metawards.mixers.merge_evenly¶

metawards.mixers.merge_evenly(network: Networks, nthreads: int, profiler: Profiler, **kwargs)¶

This merge_function merges the FOIs across all demographic sub-networks evenly. This should be equivalent to calculating the FOIs of the entire network without using demographics
