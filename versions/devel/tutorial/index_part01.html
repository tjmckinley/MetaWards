Part 1 - Modelling outbreaks¶

In this part of the tutorial you will:

  • Install metawards and perform the first model runs that simulate the outbreak of a fictional disease known as the lurgy.

  • Learn how to seed an outbreak in specific city.

  • Learn how to run a reproducible calculation.

  • Learn how to perform multiple model runs and analyse the output that is collated in the results.csv.bz2 file.

  • Discover metawards-plot and use that to make simple plots.
